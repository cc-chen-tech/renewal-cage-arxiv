# Microscopic Bilinear State-Dependent Memory Test

## Question

The rank-16 exact-force Hankel bath retained strong held-out residual memory.
This test asks whether that memory is heterogeneous dissipation controlled by
two continuous invariants of the resolved microscopic state, rather than by a
missing coordinate.

The resolved state is

```text
s_n = [v_n, z_(1,n), ..., z_(16,n)],
```

where every `z` is a training-only temporal PCA projection of 64 exact
conservative-force frames. The two rotational invariants are normalized bath
energy and velocity-first-mode power. The bilinear model is

```text
s_(n+1) = mean(s) + A0 ds_n + A_E Ehat_n ds_n
                       + A_P Phat_n ds_n + epsilon_(n+1).
```

No event label, displacement statistic, or macro observable enters the fit.

## Protocol

- system: four independent 10 tau Kob-Andersen Langevin clones at `T=0.58`;
- observations: exact force and velocity every `0.01 tau` for 64 fixed A
  particles;
- validation: leave one whole clone out;
- history: 64 frames (`0.63 tau`), rank 16;
- models: covariance-contracted stationary bath, energy-only bilinear bath,
  and energy-plus-power bilinear bath;
- primary ridge: `1e-6`, with a fixed `1e-8` to `1e-3` sensitivity range.

Promotion required mean held velocity `R2 >= 0.97`, maximum residual-state
correlation `<= 0.20`, maximum residual lag correlation `<= 0.70` and at most
80% of the stationary baseline, with improvement on every fold.

## Result

The full bilinear model raises velocity `R2` only from `0.97826` to `0.97835`
and lowers velocity residual MSE from `0.012567` to `0.012515`. It removes most
instantaneous residual-state correlation (`0.27100` to `0.02182`). However,
the decisive temporal residual correlation increases from `0.85729` to
`0.86078`, a bilinear-to-baseline ratio of `1.00407`. None of the four held
clones improves.

The ridge sensitivity does not rescue the model. From `1e-8` through `1e-4`,
the lag ratio remains above one; at `1e-3`, velocity `R2` falls to `0.79854`
and the lag correlation rises to `0.94060`.

## Interpretation

The energy and power features explain an instantaneous conditional-mean
component, but not the missing temporal state. Instantaneous residual-state
decorrelation is therefore not evidence for a closed state-dependent GLE.
The unresolved process remains colored and cannot be sampled independently in
an autonomous single-particle Langevin simulation.

The next microscopic test must introduce learned continuous cage-relative
coordinates or multiple local structural features. The bilinear bath is not
promoted to autonomous simulation:

```text
teacher_forced_state_dependent_memory_gate_pass = 0
autonomous_state_dependent_gle_allowed = 0
complete_event_clock_closure_allowed = 0
kramers_escape_claim_allowed = 0
thermodynamic_claim_allowed = 0
```
