# Smooth Force-Support Cage Projection Design

## Objective

Derive a differentiable single-particle cage coordinate directly from the
many-particle Kob--Andersen (KA) Langevin state, validate its exact projected
drift and multiplicative noise on full-particle common-noise trajectories, and
determine whether that coordinate carries a microscopic cage-escape clock.

This checkpoint is a bridge toward, but is not by itself, a closed autonomous
single-particle Langevin model. It must not fit diffusion, NGP, scattering,
event rates, or thermodynamic observables.

## Evidence motivating the design

Existing repository experiments rule out several simpler reductions:

- The instantaneous hard-neighbor cage changes neighbor count on about 46% of
  saved frames and removes all detected relative-cage events in the tested
  trajectory set.
- A fixed-neighbor cage plus Gaussian cage motion and structural marked events
  misses held-out terminal diffusion by about 49% and high-wave-number
  scattering by 61--65%.
- Scalar finite-support Volterra kernels, one-force auxiliary states, and
  unconditioned orthogonal-force autoregressions fail temporal or trajectory
  holdouts.
- Frozen-minimum and hard/hysteretic-neighbor coordinates can have nearly
  exponential isoconfigurational first-passage survival, but their tested
  structural descriptors provide weak or negative held-out committor skill.
- The C3-switched KA control validates the exact full-particle tangent-noise
  covariance over all tested intervals, so it supplies a differentiable parent
  dynamics on which a nonlinear cage projection can be tested.

The design therefore uses a continuous reaction coordinate and keeps the
state-dependent noise geometry generated by that coordinate.

## Microscopic parent dynamics

For unit particle mass, the C3-switched KA dynamics are

```text
dR = V dt,
dV = [F(R) - gamma V] dt + sqrt(2 gamma T) dW,
F(R) = -grad U_KA,C3(R).
```

All particle noises are independent Cartesian Wiener processes. The C3 radial
switch leaves the Lennard--Jones interaction unchanged below `2 sigma` and
makes `U` through `U'''` continuous at `2.5 sigma`.

## Smooth force-support cage coordinate

For tagged particle `i`, let `d_ij` be the minimum-image vector from `i` to
`j`, `r_ij = |d_ij|`, and

```text
s_ij = r_ij / (2.5 sigma_ij).
```

Use the compact Wendland C4 weight

```text
w(s) = (1-s)^6 (35 s^2 + 18 s + 3),  0 <= s < 1,
w(s) = 0,                              s >= 1.
```

The overall normalization of `w` cancels below. The support is fixed by the KA
pair-interaction support, not by an event threshold or a macro-observable fit.
Since the largest support radius is much smaller than half the simulation box,
the minimum-image branch changes only where `w` and its required derivatives
vanish.

Define

```text
W_i = sum_(j != i) w_ij,
mu_i = W_i^-1 sum_(j != i) w_ij d_ij,
C_i = r_i + mu_i,
u_i = r_i - C_i = -mu_i.
```

`C_i` is the smooth local cage center and `u_i` is the tagged displacement
relative to that center. The map is translation invariant and uses only the
instantaneous microscopic configuration and fixed KA pair parameters.

## Analytic Jacobian

Let

```text
g_ij = dw_ij/dr_ij * d_ij/r_ij,
B_ij = [w_ij I + (d_ij-mu_i) g_ij^T] / W_i.
```

Then

```text
du_i = J_ii dr_i + sum_(j != i) J_ij dr_j,
J_ii = sum_(j != i) B_ij,
J_ij = -B_ij.
```

The translational identity is exact:

```text
J_ii + sum_(j != i) J_ij = 0.
```

The cage-relative velocity is therefore

```text
p_i = du_i/dt = J_i(R) V
    = sum_(j != i) B_ij (v_i-v_j).
```

The implementation returns sparse-local blocks in a dense particle-aligned
array so that contractions with the full force and noise vectors are explicit
and testable.

## Exact instantaneous projected SDE

Because `R` has finite variation in underdamped Langevin dynamics, Ito's rule
gives no second-order correction in `du_i`. Differentiating `p_i=J_iV` gives

```text
du_i = p_i dt,

dp_i = b_i(R,V) dt + sqrt(2 gamma T) J_i(R) dW,

b_i(R,V) = J_i(R) F(R)
             + Hess(u_i;R) : (V tensor V)
             - gamma p_i.
```

The Hessian-velocity contraction is evaluated as the centered directional
derivative

```text
Hess(u_i;R):(V tensor V)
  = [J_i(R+hV)V - J_i(R-hV)V] / (2h) + O(h^2).
```

Only particles in the smooth support have nonzero Jacobian blocks, so `J_iF`
requires conservative forces for the tagged particle and its supported
neighbors, not an all-pairs force matrix.

Define

```text
G_i(R) = J_i(R) J_i(R)^T,
M_i(R) = G_i(R)^-1.
```

Then the conditional projected-noise covariance is

```text
Cov[dp_i | R,V] / dt = 2 gamma T G_i(R).
```

Multiplication by `M_i` gives a generalized force equation whose thermostat
friction and noise obey the instantaneous matrix FDT:

```text
M_i dp_i = M_i[b_i + gamma p_i] dt - gamma M_i p_i dt
           + sqrt(2 gamma T) M_i J_i dW,

Cov[sqrt(2 gamma T) M_i J_i dW] / dt = 2 gamma T M_i.
```

This identity is exact but not autonomous because `b_i`, `G_i`, and the
orthogonal future still depend on the eliminated many-particle state.

## Common-noise tangent validation

For matched `+epsilon` and `-epsilon` full-particle paths driven by the same
Wiener stream, define central responses

```text
delta p = (p+ - p-) / (2 epsilon),
delta b = (b+ - b-) / (2 epsilon),
delta J = (J+ - J-) / (2 epsilon).
```

Their tangent SDE contains the multiplicative projected-noise term

```text
d(delta p) = delta b dt + sqrt(2 gamma T) delta J dW,

Cov[d(delta p)-delta b dt | paths] / dt
  = 2 gamma T delta J delta J^T.
```

The integrated covariance uses the same trapezoid/stride convention as the
validated C3 force-generator tangent experiment. No interval is removed merely
because a particle enters or leaves support: the compact weight is smooth at
the support boundary.

## Null models

The diagnostic reports the following controls without changing their
definitions in response to results:

1. `raw_tagged`: `u=r_i`. Its Jacobian is constant and the common-noise
   tangent multiplicative covariance is zero.
2. `fixed_neighbor`: the mean displacement of initial neighbors inside
   `1.5 sigma`. Its Jacobian is constant and it cannot update cage membership.
3. `smooth_force_support_without_delta_J`: the proposed coordinate with its
   multiplicative tangent covariance deliberately set to zero.
4. `hard_dynamic_neighbor`: the existing instantaneous cutoff coordinate,
   retained only to quantify graph discontinuities and event loss.

## Preregistered gates

### Coordinate gates

- Exact decomposition error `max|tagged-cage-relative| <= 1e-12` on synthetic
  and recorded configurations.
- Translation-invariance error `max|sum_j J_ij| <= 1e-12`.
- Analytic-Jacobian finite-difference relative L2 error `<= 1e-6` on random
  non-boundary configurations.
- Velocity identity relative L2 error `<= 1e-6` against a centered
  configuration-time derivative at decreasing saved-frame stride.
- Every reported `G_i` is positive definite with condition number recorded.

### Tangent-SDE gates

- All 8 members and both `epsilon=0.001,0.002` are analyzed on the common
  201-frame C3 horizon.
- Cross-epsilon integrated-covariance correlation `>= 0.995` and maximum
  member relative L2 difference `<= 0.05`.
- At the selected preregistered stride, pooled covariance trace ratio is in
  `[0.8,1.2]` for both epsilons.
- Mean squared Mahalanobis value is in `[2.4,3.6]` for both epsilons, against
  the three-dimensional reference value 3.
- Absolute pooled whitened lag-1 correlation is `<= 0.1` for both epsilons.
- If the gates fail at every tested stride, the projected multiplicative-noise
  closure is rejected rather than retuned through a new cage weight.

### Event-clock gates

After the tangent checkpoint, apply the unchanged nonrecrossing `p_hop`
protocol to long KA trajectories:

- Use one weight definition at all parents and temperatures.
- Report first-passage censoring and survival curves before fitting a rate.
- Require leave-one-parent-out held-out Brier skill greater than zero for a
  hazard based only on microscopic `(u,p,G,b)` state.
- Compare against the existing best tested relative-neighbor structural
  committor skill, about `0.027`; failure to exceed it means this coordinate is
  dynamically exact at short times but not a sufficient escape coordinate.
- Only after a positive event-clock result may a conditional memory kernel and
  orthogonal-force simulator be promoted to the next phase.

## Claim boundary

A passing tangent experiment establishes an exact microscopic instantaneous
single-particle cage-coordinate SDE and its state-dependent FDT geometry. It
does not establish an autonomous GLE, a renewal hazard, Kramers escape, held-out
diffusion/NGP/scattering closure, configurational entropy, a Kauzmann
temperature, or an ideal thermodynamic glass transition.

`thermodynamic_claim_allowed = 0` remains mandatory in every output.
